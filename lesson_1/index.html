Different types of programming

You need to understand the different types of programming there is, the most basic web development, then mobile app development, then desktop application, but don't understimate it, it can become very complex even at web development depending on how far you will take it.

Introduction to web development

Web development is about making websites, and it uses programming languages at its core HTML, CSS and Javascript and they have different functions in making the website, HTML is the structure of the website, CSS is the design of the website and Javascript is the interactivity, logic and brain of the website. There is also backend programming language that are used when you need to store data and used data at your website, like for example banking apps that need to store your data and you can make transactions.

Hints and Tips

Design is overlooked by many beginners but it is very important to learn the basics of design and how to make a good design, it is not just about making a website that works but also making a website that looks good, There are two different routes you can learn design and programming seperately or you can learn and combine them. But there is a certain level of design and graphic design you need to learn, even if you want to learn programming only. Because your projects or application or website need to look good and design, you need to have control and comfortablility of design. Because if your making a portfolio website to show your projects, or making a product you need to have a good design and control and comfortability with design, so you need to learn enough to achieve that.
