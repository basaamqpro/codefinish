What is HTML, CSS, and JS

HTML stands for HyperText Markup Language, it is the structure or blueprint of the website, you can add images, text, videos and shapes. CSS stands for Cascading Style Sheets, it's how you design the structure of shapes, text, images and videos of the structure of HTML. JS stands for Javascript, it is the language the true core programming language of the of website developments you will be doing.

HTML

You just have to learn these codes to become a web developer, we will learn them now generally and then we will break them down each one in the next lessons.

<p></p>

<div></div>

<h1></h1>

<a href="#">Link</a>

<img src="image.jpg">

<video src="video.mp4" controls></video>

<audio src="audio.mp3" controls></audio>

The first three elements that you write <div>, <h1>, <a>, can contain content inside them.

If you observe closely, you have to close these elements like: <element></element>

The <img> element is a self-closing (void) element.

CSS

CSS has more code, but it's easier to understand because it's readable. It is divided into properties and values.

I'll show you some properties.

background-image: url('image.jpg');

This will give a background image for an element, which can be a div, section, header, or many other HTML elements.

background-image: linear-gradient(45deg, red, blue);

This will create a background with colours blended together in a gradient.

font-family: sans-serif;

This changes the font used by the text. It doesn't have to be sans-serif; there are thousands of fonts available.

font-size: 20px;

The font-size property allows you to increase or decrease the size of text using units such as pixels (px).

color: red;

The color property allows you to change the colour of text, for example text inside a <p> tag.

margin: 4px 8px 16px 20px;

The margin property allows you to add space around an element.

The values represent: Top = 4px, Right = 8px, Bottom = 16px, Left = 20px.

padding: 4px 8px 16px 20px;

The padding property allows you to add space inside an element.

The values represent: Top = 4px, Right = 8px, Bottom = 16px, Left = 20px.

The difference between margin and padding is that margin adds space outside an element, while padding adds space inside it.

position: fixed;

top: 50%;

left: 60%;

The position property with the value fixed works together with top and left to position an element on the screen.

Remember that top moves the element down from the top edge of the screen.

For example, top: 50%; moves the element halfway down the screen.

left: 60%; moves the element 60% from the left side of the screen.

display: flex;

justify-content: center;

align-items: center;

We will learn more about Flexbox later. It's a powerful way of arranging and positioning elements.

object-fit: fill;

object-fit: cover;

object-fit: contain;

This controls how an image or video is resized inside its container.

Depending on the value, the image can stretch, fit perfectly, or be cropped.

@media only screen and (orientation: landscape)

@media only screen and (orientation: portrait)

Media queries allow your website to react to different screen sizes and orientations.

Portrait is usually associated with mobile devices because the height is greater than the width.

Landscape is usually associated with desktop screens because the width is greater than the height.
